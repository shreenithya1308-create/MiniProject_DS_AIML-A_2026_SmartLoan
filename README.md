# SmartLoan: Predicting Microfinance Loan Default Risk

## 📌 Abstract
This project focuses on predicting loan default risk using data science techniques. By analyzing applicant information such as income, loan amount, credit history, and demographic details, the model identifies patterns associated with loan approval or rejection. Data preprocessing and visualization were performed to understand the dataset. A Logistic Regression model was trained to classify loan status, achieving an accuracy of approximately 79%. The project demonstrates how machine learning can assist financial institutions in minimizing risk.

---

## ❗ Problem Statement
Microfinance institutions face challenges in identifying borrowers who may default on loans. This project aims to build a predictive model to classify applicants as low-risk or high-risk based on their financial and personal attributes.

---

Loan Prediction Dataset (Kaggle)

## 📈 Results
- Accuracy: ~79%  
- Model performs better in predicting loan approvals than rejections  
- Credit history is the most important factor  

---

## 📌 Conclusion
The model successfully predicts loan approval with good accuracy. However, it performs better for majority class (approved loans). Future improvements can include handling class imbalance and using advanced models.
