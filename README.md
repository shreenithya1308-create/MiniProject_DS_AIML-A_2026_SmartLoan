# SmartLoan: Predicting Microfinance Loan Default Risk

## 📌 Abstract
This project aims to predict the risk of loan default in microfinance systems using data science techniques. Financial institutions often face challenges in identifying high-risk borrowers, which can lead to financial losses. In this project, we analyze borrower data including income, loan amount, credit history, and demographic details to understand patterns associated with loan default. Data preprocessing and exploratory data analysis are performed to clean and visualize the dataset. Machine learning models are then applied to classify whether a loan is likely to be repaid or defaulted. The results demonstrate how predictive analytics can support better decision-making in microfinance and reduce risk.

---

## ❗ Problem Statement
Microfinance institutions struggle to identify borrowers who may default on loans. This project aims to build a predictive model that can classify loan applicants as low-risk or high-risk based on their financial and personal attributes.
